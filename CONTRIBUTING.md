# Contributing

Thank you for helping grow this list!

## Adding a Site

Your site qualifies if it:

- Serves a `skill.md` at a public URL (e.g. `https://example.com/skill.md`)
- The `skill.md` follows [OpenClaw skill format](https://docs.openclaw.ai/tools/skills) (YAML frontmatter + instructions)
- The described API/endpoints are publicly accessible

## Entry Format

```markdown
- [Site Name](https://example.com/) - Brief description ending with a period.
```

- Use a clear, concise description (~10-15 words).
- No promotional language.
- Place the entry **alphabetically** within the correct category.

## How to Submit

1. Fork this repo.
2. Add your site to the README following the format above.
3. One site per pull request.
4. Make sure your `skill.md` is publicly accessible.
5. Submit a PR.
